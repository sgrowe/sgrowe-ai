---
name: plan-then-handoff-implementation
description: Plan work, and then hand it off to a teammate to implement
---

Create a detailed implementation plan (in `.scratch/plans/`) and then spawn a teammate to handle the implementation

Once the implementation is done, spawn two more teammates in parallel to provide feedback to the first. Spawn one with the `/verify` skill to QA the work, and spawn the other with the `/code-review` skill to provide code review feedback
